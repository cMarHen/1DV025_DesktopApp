# &lt;flipping-tile&gt;

Astroid shooter main application. 

## Methods

### `startGame()`

Will start the game.

Parameters: None.

Returns: None.

### `setToStorage()`

Set data to local storage.

Parameters: None.

Returns: None.

## Events

| Event Name | Fired When |
|------------|------------|
| `username-set`| When a username is chosen.
| `clicked`| Button PLAY are clicked.
| `game-over`| Astroid field is game over.
